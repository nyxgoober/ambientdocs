# Ambient Documentation

This repository contains the source code for the HTML file of the **Ambient Wiki**.

## Contributing

Contributions are welcome.  
If you’d like to improve the project, please submit changes via a **Pull Request**.

Before contributing, please ensure your changes are:
- Clear and well-documented
- Tested where applicable
- Aligned with the overall structure and goals of the project
- In the correct section of the wiki

We appreciate all improvements, whether they are new documented bugs, API documentation or wording improvements.
